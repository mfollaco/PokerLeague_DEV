LUCK PAGE FINDINGS

1. frontend/js/analytics/luck.js is functional but architecturally different from newer analytics pages.
2. luck.js currently owns:

   - season JSON loading
   - math helpers
   - page skeleton rendering
   - chart rendering
   - table rendering
   - sorting behavior

   - page initialization
3. This makes luck.js more self-contained and less aligned with the shared core/page_bootstrap pattern used elsewhere.
4. Recommended classification:
   - keep as-is for now
   - refactor later
